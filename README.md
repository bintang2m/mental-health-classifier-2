# Mental Health Classifier

Website untuk klasifikasi kesehatan mental berbasis teks menggunakan model `B1NT4N9/roberta-mental-health-id` dari Hugging Face.

## Fitur
- ✅ Klasifikasi 5 kategori: Normal, Anxiety, Depression, Bipolar, Suicidal
- ✅ Analisis teks bahasa Indonesia
- ✅ Threshold adjustment
- ✅ Browser-based processing
- ✅ Responsive design dengan Bootstrap 5
- ✅ Native JavaScript (no frameworks)
